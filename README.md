# ML Dowscaling Emulation Data


## Installation

Assumes you have conda (or equivalent like mamba installed).

1. Clone repo and cd into it
2. Create conda environment: `conda env create -f environment.lock.yml` or `conda install --file environment.txt` (depending on whether you want to create a new environment based on exact version developed on or install any necessary conda packages in your own environment)
3. Activate the conda environment (if not already done so)
4. Install package: `pip install -e .`
